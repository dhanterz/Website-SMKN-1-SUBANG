Rekayasa Perangkat Lunak (RPL)

Tentang RPL

Rekayasa Perangkat Lunak (RPL) adalah program keahlian yang berfokus pada pengembangan perangkat lunak, pemrograman, dan sistem informasi. Siswa akan belajar tentang berbagai bahasa pemrograman, pembuatan aplikasi, serta analisis dan desain sistem.

Materi yang Dipelajari

  • Algoritma dan Pemrograman
  • Basis Data
  • Pemrograman Web
  • Pemrograman Mobile
  • Desain UI/UX
  • Analisis dan Perancangan Sistem
  • Jaringan dan Keamanan
  • Manajemen Proyek IT

Prospek Karir Rekayasa Perangkat Lunak

Lulusan RPL memiliki peluang kerja yang luas di bidang teknologi informasi, termasuk:

  • Software Developer
  • Web Developer
  • Mobile App Developer
  • UI/UX Designer
  • Database Administrator
  • System Analyst
  • Cyber Security Specialist
